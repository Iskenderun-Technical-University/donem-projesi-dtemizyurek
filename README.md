# MovieApp

## Projenin Amacı

Günümüzde bir çoğumuz film izlemekte ve izlediği filmleri çoğu zaman unutmakta. Bu proje ile izlediğimiz filmleri ve açıklamalarını ufak bir kütüphane şeklinde bulabilmekteyiz.
